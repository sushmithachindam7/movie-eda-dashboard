# Movie EDA Dashboard

A Streamlit web app for performing **Exploratory Data Analysis (EDA)** on movie datasets.  
## Features
- **Descriptive Analysis**: Summary statistics, genre distribution.
- **Diagnostic Analysis**: Correlation heatmap, ratings by genre.
- **Predictive Analysis**: Linear regression model to predict revenue from budget.
- **Prescriptive Analysis**: ROI analysis recommending best genre and decade for investment.
